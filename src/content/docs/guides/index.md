ScorpioBot is a Discord bot built with modularity in mind. Is serves as a base for users to build upon and extend using
both custom-made and community-created modules.

For list of publicly available modules, check out [Community Resources](#) page.

## Why ScorpioBot?
While there are a lot of available bots on the market, we believe that our resource stands out from the crowd with:
1. **FOSS mission**
2. **Unique set of features**
3. **Developer-first growth**

You can read more about our mission below.

## Free and Open Source
We believe that everyone deserves a good option for handling their Discord community regardless if they are a small 
server of big community. That's why ScorpioBot was created as a free and open resource for everyone to enjoy and contribute to.

## Unique set of features
While are a lot of different approaches to Discord bots, ScorpioBot focuses on creating unique set of features that make 
everyone's life easier.

These features include [dynamic translation system](#) for communities who aggregate people using different languages
and [multiple developer-focused tools](#) which also allow for more customization by users.

## Developer-first growth
Our main focus is a balance between good user and developer experience. That's why every part of ScorpioBot allows to
easily create modules with maximized possibilities of customization like messages, commands and other utilities.
We want to continue this development path by adding more features without making the bot hard to use by a typical user
which isn't always tech-savy.
